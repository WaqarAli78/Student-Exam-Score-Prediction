🎓 Student Exam Score Predictor



A machine learning-powered web application built with Streamlit that predicts a student's exam score based on study habits, attendance, sleep patterns, mental health, and part-time job status.

🚀 Features

✅ Interactive Streamlit Interface
✅ Real-time Exam Score Prediction
✅ Simple and User-Friendly Controls
✅ Trained Machine Learning Model (best_models.pkl)
✅ Deployable on Streamlit Community Cloud

🧠 Input Features
Feature	Description
Study Hours	Hours spent studying per day
Attendance	Attendance percentage
Mental Health	Rating from 1–10
Sleep Hours	Average sleep hours per night
Part-Time Job	Yes or No
⚙️ Installation

Clone the repository

git clone https://github.com/WaqarAli78/Student-Exam-Score-Prediction.git

Move into the project directory

cd StudentExamPredictor

Install dependencies

pip install -r requirements.txt

Run the Streamlit application

streamlit run App.py
